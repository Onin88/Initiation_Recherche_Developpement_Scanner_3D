# Compilation
Dans chaque dossier correspondant à un type de scan, il y a des fichiers python executables. Des modules pip sont nécessaires, à installer selon les erreurs.
> [!IMPORTANT]
> ```sh
> python <nomDuFichier.py>
> ```
